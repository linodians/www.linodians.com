A JSON alternative to RSS and Atom, JSON Feed is web syndication file format.
It is a way for feed readers, including humans, to stay connected with blogs, podcasts, social networks, and more.
JSON Feed was created by Manton Reece and Brent Simmons in 2017.
Linodians has provided a section on this website in order to show support to and grow the adoption of JSON Feed.
