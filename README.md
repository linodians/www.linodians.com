# Linodians Website ([www.linodians.com](https://www.linodians.com))
[![GitHub License](https://img.shields.io/badge/license-MIT-blue.svg)](https://raw.githubusercontent.com/linodians/www.linodians.com/trunk/LICENSE)

This is the public repository for <https://www.linodians.com>, a static website generated by [Hugo](https://github.com/gohugoio/hugo).


## Run Locally

You can run this website locally by cloning this repo and serving it with Hugo:

```
git clone https://github.com/linodians/www.linodians.com.git
cd www.linodians.com
hugo -s src serve --open
```

It'll then open in your default browser at `http://localhost:1313/`.

## Contributing

### Glossary

To add a new glossary term, start by creating the file:

```bash
hugo -s src new content --kind=glossary docs/reference/glossary/<term-slug>.md
```

Then edit the file, filling in the information.
